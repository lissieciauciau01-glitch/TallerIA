Prompt listo: ¿Qué es un commit y por qué es importante?

Este archivo contiene la respuesta estilizada y un prompt listo para copiar e insertar en tu HTML. Usa el botón para copiar el texto completo del prompt.

Vista
  • Definición: Un commit es una instantánea del estado del repositorio que guarda cambios, mensaje y metadatos.
  • Importancia: historial, reversión, colaboración, trazabilidad y mejores prácticas.
  • Buenas prácticas: commits atómicos, mensajes en imperativo, probar antes de commitear, usar `.gitignore`.
Comandos básicos 
git status
git add .
git commit -m "Mensaje claro"
git commit -am "Mensaje"
git log
git checkout -b rama-nueva
git push origin rama-nueva
git revert <hash>
